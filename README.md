## 🚀 Quick Start

1. Clone the repository:

   ```bash
   git clone https://github.com/PaleEXE/pale-uni-app.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   uvicorn main:app --reload
   ```
